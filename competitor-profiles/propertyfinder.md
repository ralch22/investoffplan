## Product

UAE's dominant property portal. Off-plan lane: unit-level SERP (`view=unit_types`) + developer/project PDPs with rich editorial content, payment plans, timelines, maps, and resale/sold-history data.

## SERP strengths

- Default unit-level view with ~1,530 unit cards
- Each card: hero image, unit count badge, property type, launch price range, premium flag, developer, city/area, beds, sqft
- Strong brand trust and mobile-native filtering

## PDP strengths (Address Residences Zabeel sample)

- Launch price hero
- Key information: delivery, sale start, location, payment plans, buildings, types, gov fee, ownership
- Payment plan explorer
- 4-step project timeline
- Long-form HTML about + masterplan image
- Location map + "Living in [area]" community module with scores/reviews
- Full amenities list
- Sold history table + resale inventory count
